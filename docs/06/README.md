按照「艾塔索尔太阳历」，这是第四十七个千禧年。

若照「格林斯－古多瓦」纪年，则是天轮金耀六百三十纪。大转盘跃过摩羯，开启阿古硫斯的宝瓶时代。

「我的孩，每逢时代进递或光轮回转，黑色的恶必会降临。它是必来的，它必与蛇结盟。凡它所到的万国，必要流血。分明在高处的就要陨落；分明有眼目的就不能看见；分明有手脚的就要受煎熬……」

……

朱颜血第六颗红泪，于焉坠落！
